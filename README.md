# 📄 PDF Question Answering Tool using BERT  
  
A simple **Extractive Question Answering** tool that reads a PDF document and answers natural language questions about its content using a pre-trained **BERT** transformer model fine-tuned on SQuAD.  
  
---  
  
## 🚀 Features  
  
- Extracts text from any PDF file  
- Answers questions using `bert-large-uncased-whole-word-masking-finetuned-squad`  
- Handles long documents via text chunking  
- Returns answer confidence scores  
- Runs seamlessly on **Google Colab**
